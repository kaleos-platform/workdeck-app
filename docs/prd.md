# 쿠팡 광고 리포트 매니저 MVP PRD

## 🎯 핵심 정보

**목적**: 쿠팡 광고 리포트 Excel을 업로드·분석해 비효율 광고비를 절감하고 ROAS를 개선한다
**사용자**: 쿠팡 광고를 직접 운영하는 1인 셀러 또는 소규모 쇼핑몰 운영자

---

## 🚶 사용자 여정

```
1. 회원가입 페이지
   ↓ 이메일·비밀번호 입력 후 가입 완료

2. 워크스페이스 설정 페이지 (최초 1회)
   ↓ 쿠팡 사업자명 입력 후 생성

3. 대시보드 (전체 워크스페이스 요약)
   - 업로드 데이터 없음 → 리포트 업로드 CTA 표시
   ↓ 업로드 버튼 클릭

4. 파일 업로드 페이지
   - Excel(.xlsx) 파일 선택 또는 드래그앤드롭
   - 파싱 → DB 저장 → NVB 캠페인 목록 갱신
   ↓ 업로드 성공

5. 캠페인 상세 페이지 (NVB에서 캠페인 선택)
   - [대시보드 탭]    → 성과 KPI 카드 + 시계열 그래프 + 일자별 메모
   - [광고 데이터 탭] → 원시 데이터 테이블 (정렬·필터)
   - [키워드 분석 탭] → 비효율 키워드 목록 + 다중 선택 + 클립보드 복사
   ↓ 각 탭 상단의 공통 필터 조작

6. 날짜 범위·광고유형 필터
   - 모든 탭에 공통 적용, 필터 변경 시 해당 탭 데이터 즉시 갱신
```

## ⚡ 기능 명세

### 1. MVP 핵심 기능

| ID | 기능명 | 설명 | MVP 필수 이유 | 관련 페이지 |
|----|--------|------|-------------|------------|
| **F001** | 광고 리포트 업로드 | Excel(.xlsx) 파싱 후 DB 저장, NVB 캠페인 목록 갱신 | 모든 분석의 데이터 소스 | 파일 업로드 페이지 |
| **F002** | 시계열 성과 그래프 | 날짜별 ROAS·광고비·클릭수·노출수 추이 Recharts 차트 | 광고 성과 트렌드 파악 | 캠페인 상세 페이지 - 대시보드 탭 |

| **F004** | 광고 데이터 테이블 | 캠페인·키워드 단위 컬럼 정렬·필터·페이지 크기 설정 테이블 | 데이터 탐색 및 비교 | 캠페인 상세 페이지 - 광고 데이터 탭 |
| **F005** | 일자별 메모 | 날짜 선택 후 광고 작업 내용 CRUD 메모 | 튜닝 이력 추적 | 캠페인 상세 페이지 - 대시보드 탭 |
| **F006** | 날짜 범위 필터 | 날짜 피커로 분석 기간 설정, 각 탭 공통 적용 | 기간별 비교 분석 | 캠페인 상세 페이지 (모든 탭) |
| **F007** | 광고유형 필터 | 드롭다운으로 광고유형 선택, 각 탭 공통 적용 | 광고 단위별 관리 | 캠페인 상세 페이지 (모든 탭) |

|------|------|----------|

## 🛠️ 기술 스택

### 🎨 프론트엔드 프레임워크

- **Next.js 16** (App Router, RSC) - React 풀스택 프레임워크, 기존 적용
- **TypeScript 5** - 타입 안전성 보장, 기존 적용
- **React 19** - UI 라이브러리 (동시성 기능 활용), 기존 적용

### 🎨 스타일링 & UI

- **Tailwind CSS v4** (설정 파일 없는 새로운 CSS 엔진) - 기존 적용
- **shadcn/ui** (new-york 스타일) - 고품질 React 컴포넌트 라이브러리, 기존 적용
- **Lucide React** - 아이콘 라이브러리, 기존 적용

### 📝 폼 & 검증

- **React Hook Form 7.x** - 폼 상태 관리, 기존 적용
- **Zod** - 스키마 검증 라이브러리, 기존 적용

### 📊 차트 & 시각화

- **Recharts** - 시계열 라인/바 차트, 기존 적용

### 🗄️ 백엔드 & 데이터베이스

- **Supabase** - Auth 및 세션 관리 (SSR 방식), 기존 적용

- **PostgreSQL** - 관계형 데이터베이스 (Supabase 포함), 기존 적용

### 📦 클라이언트 상태 관리
