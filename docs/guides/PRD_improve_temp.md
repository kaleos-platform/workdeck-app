# 쿠팡 광고 관리자 개선 작업

## Supabase 계정 인증 메일 개선

1. 인증 메일에서 인증 확인 시 랜딩 페이지를 로그인 화면으로 수정

## 대시보드 화면 개선(/dashbaord)

1. 기간 설정 영역 추가

- 캠페인 상세화면의 기간 설정 기능을 대시보드에 추가한다
- 기본 설정값은 7일로 설정한다.
- 기간을 변경하면 해당 기간에 맞춰서 지표 정보가 변경된다.

2. 대시보드 페이지의 캠페인 목록의 정보를 개선한다

- 1에서 설정한 기간에 맞춰 해당 캠페인 별로 지표 정보를 표시한다.
- 지표는 "총 광고비", "평균 ROAS", "총 매출액" 까지 표시한다.
- 동일하게 설정된 기간에 맞춰 이전 기간 대비 증감 여부도 같이 표시한다.

## 캠페인 상세 화면 개선(/dashboard/campaigns/)

1. 키워드 분석 탭 개선

- 광고비 지출&주문수 0인 키워드만 표시하지 않고 모든 키워드를 표시
- 광고비 지출&주문수 0인 키워드는 필터 옵션으로 추가해서 체크 시 해당 조건의 키워드만 표시하도록 수정
- 모든 키워드 표시할 때 기본 정렬 순서는 광고비 내림차순
- 주문 발생 키워드 보기 필터 옵션 추가: 주문 건수가 1개 이상인 키워드만 노출

## 캠페인 별 일 예산 및 목표 ROAS 설정 기능

1. 캠페인 별로 일 예산와 목표 ROAS를 설정할 수 있습니다.

- 캠페인의 일 예산과 목표 ROAS를 수동으로 입력하고 수정할 수 있도록 해주세요.
- 캠페인 상세화면에서 일 예산 또는 목표 ROAS가 입력되지 않은 경우에 입력할 수 있도록 사용자가 직관적으로 인지하고 입력을 유도할 수 있는 UI를 고려해주세요.
- 일 예산 또는 목표 ROAS가 설정되면 /dashboard 화면에서 캠페인 목록에 해당 정보를 표시해주세요.
- 일 예산과 목표 ROAS는 일자 단위로 관리가 되어야 합니다.
- 예산과 ROAS를 등록할 때 이전 일자에 등록된 값이 있을 경우 등록 시점부터 이전일자까지 이전 기간의 값이며 변경일자부터는 변경된 값으로 등록됩니다.
- 모든 정보는 캠페인에 등록된 광고 집행일 기준으로 관리되어야 합니다.
- 일 예산과 목표 ROAS 변경/등록 시 캠페인의 광고 잡행일자 기간 내에서 이전 일자에 해당 값을 필요할 경우 직접 수정할 수 있습니다.

2. 캠페인 상세화면에 일 예산 평균 소진율 지표 추가

- 캠페인 상세화면에 "일 예산 평균 소진율"을 추가합니다.
- "일 예산 평균 소진율"은 설정된 기간 동안에 예산이 있을 경우 해당 기간 동안 일일 광고 집행액 합계/일 예산 \* 100 (% 표시, 소수점 2자리까지 표시)의 평균값으로 표시됩니다.
- 만약 설정한 기간 동안의 일 예산이 없을 경우 "일 예산 평균 소진율"은 표시하지 않고 "해당 기간에 설정된 예산이 없다고 안내합니다."

3. 캠페인 상세화면에 목표 ROAS 달성율 지표 추가

- 캠페인 상세 화면에 "목표 ROAS 평균 달성율"을 추가합니다.
- "목표 ROAS 달성율"은 설정된 기간 동안에 목표 ROAS가 있을 경우 해당 기간 동안 일 광고 ROAS / 목표 ROAS \* 100 (%표시, 소수점 2자리까지 표시)의 평균값으로 표시됩니다.
- 만약 설정한 기간 동안의 일 예산이 없을 경우 "목표 ROAS 달성율 지표"는 표시하지 않고 "해당 기간에 설정된 목표 ROAS가 없다고 안내합니다."

4. 2,3의 일 예산 평균 소진율과 목표 ROAS 달성율 지표는 대시보드 탭에 표시해주세요.

- 표시할 때 기존의 지표(총 광고비, 총 매출액, 평균 ROAS, 평균 CTR, 평균 CVR) 위에 다른 계위로 표시해주세요.
- 해당 지표는 현재 캠페인의 상태를 나타내는 지표입니다. 광고를 잘 관리하는지를 가이드해주는 컴포넌트 내에서 지표가 표시되도록 UIUX를 구성해주세요.

5. 일 예산과 목표 ROAS 변경 시 메모 추가

- 일 예산과 목표 ROAS를 변경하고 저장한 후에 메모 저장 여부를 확인하는 팝업창을 표시해주세요.
- 메모를 표시하는 방법은 아래와 같습니다.
  - "일 예산 변경: {이전 예산} -> {변경 예산}"
  - "목표 ROAS 변경: {이전 목표 ROAS} -> {변경 후 목표 ROAS}"
- 팝업 창에서 기본으로 오늘 날짜가 선택되어서 메모 저장 시 오늘 날짜로 저장해주세요.
- 팝업 창에서 설정된 날짜를 변경해서 메모를 다른 날짜에 입력할 수 있습니다.
