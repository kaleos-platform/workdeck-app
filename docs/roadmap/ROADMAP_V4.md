# 쿠팡 광고 리포트 매니저 개발 로드맵 V4

쿠팡 광고 리포트 매니저의 사용자 경험(UX)을 개선하고, 데이터의 가시성을 높이며, 운영 중 발견된 버그를 수정하여 서비스의 안정성을 강화합니다.

## 개요

PRD V4에서는 다음 핵심 가치를 실현하기 위한 기능을 제공합니다:

- **데이터 가시성 강화**: 캠페인 목록에서 데이터 업로드 기간을 즉시 확인하고 누락된 데이터를 파악 (F038)
- **UI/UX 최적화**: 캠페인 목록 및 상세 페이지의 설정 섹션을 직관적으로 개편 (F034, F037)
- **운영 효율성 향상**: 메모 일자 수정 기능 제공 및 키워드 필터링 로직 고도화 (F033, F035)
- **서비스 안정화**: 탭 전환 시 발생하는 UI 버그 수정 (F036)

## 개발 워크플로우

1. **작업 계획**
   - 기존 코드베이스와 PRD V4 요구사항을 대조하여 영향 범위 분석
   - 새로운 작업을 포함하도록 `ROADMAP_V4.md` 업데이트

2. **작업 생성**
   - `/tasks` 디렉토리에 새 작업 파일 생성 (예: `033-memo-popup-bugfix.md`)
   - 각 작업 파일에 고수준 명세, 수락 기준, 구현 단계, **테스트 체크리스트(Playwright MCP)** 포함

3. **작업 구현**
   - **구조 → UI → 기능** 순서로 개발 진행
   - API 연동 및 비즈니스 로직 구현 시 Playwright MCP를 활용한 E2E 테스트 필수 수행
   - 각 단계 완료 후 작업 파일 및 로드맵 업데이트

4. **로드맵 업데이트**
   - 로드맵에서 완료된 작업을 ✅로 표시하고 관련 작업 파일 링크 추가

## 개발 단계

### Phase 1: 구조 보강 및 데이터 인터페이스 설계

- **Task 033: API 응답 타입 확장 및 유틸리티 보강** - 우선순위
  - `CampaignWithMetrics` 타입에 `minDate`, `maxDate` 필드 추가
  - `src/lib/date-range.ts`의 KST 날짜 변환 유틸리티 점검
  - 메모 수정 모드 구분을 위한 `MemoDialog` Props 인터페이스 수정

### Phase 2: UI/UX 개편 (레이아웃 및 정적 컴포넌트)

- **Task 034: 캠페인 목록(Dashboard) UI 개선 (F037, F038)**
  - 캠페인 카드 내 지표(광고비, ROAS, 매출)와 증감 수치를 한 줄로 배치
  - 데이터 입력 기간 표시부 및 "업로드 필요" 경고 UI 구현
  - 일 예산/목표 ROAS 텍스트 스타일 가독성 개선

- **Task 035: 캠페인 상세 설정 섹션 카드 분리 (F034)**
  - "광고 관리 현황" 카드에서 지표 섹션과 설정 섹션을 별도 카드로 분리
  - "일 예산 / 목표 ROAS" 신규 카드 레이아웃 구현 (이력 보기 버튼 포함)

- **Task 036: 키워드 분석 필터 및 탭 UI 개선 (F035)**
  - 필터 버튼명 변경 및 "제거 제외" 토글 UI 추가
  - 탭 간 이동 시 상태 유지/초기화 로직 준비

### Phase 3: 핵심 비즈니스 로직 및 기능 구현

- **Task 037: 캠페인 목록 데이터 기간 API 연동 (F038)**
  - `/api/campaigns` 라우트에서 `groupBy`를 활용한 캠페인별 날짜 범위 집계 로직 구현
  - KST 기준 날짜 복원 및 클라이언트 사이드 업로드 필요 판단 로직 연동

- **Task 038: 메모 시스템 고도화 및 버그 수정 (F033, F036)** - 우선순위
  - 탭 전환 시 `memoTarget` 초기화 로직 구현 (F036 버그 수정)
  - 메모 일자 수정 시 기존 메모 삭제 및 병합(Upsert) 로직 구현 (F033)
  - Playwright MCP를 활용한 메모 병합 시나리오 검증

- **Task 039: 설정 변경 이력 팝업 및 관리 기능 (F034)**
  - 이력 조회를 위한 Dialog 팝업 구현
  - 이력 목록 테이블 연동 및 개별 이력 삭제/수정 기능 구현

### Phase 4: 안정화 및 최종 검증

- **Task 040: V4 통합 테스트 및 성능 최적화**
  - Playwright MCP를 사용한 전체 사용자 시나리오(F033~F038) E2E 테스트
  - 필터 선택 시 자동 정렬 기능(F035) 및 탭 전환 안정성 최종 검증
  - 빌드 테스트 및 배포 전 최종 점검
