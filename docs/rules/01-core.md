# Core Rules

- scope: global
- priority: 100
- strength: MUST

## 1. Language

- 기본 응답: 한국어, 간결하게.
- 코드 주석: 한국어.
- 커밋 메시지: Conventional Commits + 한국어 설명.
- 문서화: 한국어.
- 변수/함수명: 영어 + 코드 표준 준수.

## 2. Code Quality

- SRP 준수, 모듈화 우선.
- 최신 권장 패턴 사용.
- any 사용 지양 (명시적 타입 우선).
- 모든 비동기 로직에 에러 핸들링 포함.

## 3. Response Behavior

- 추측 금지, 불확실하면 질문.
- 불필요한 서론/감정 표현 금지.
- 구현 전 3–7개 불릿 계획 작성.
- 규칙 위반 가능성 있으면 명시.
